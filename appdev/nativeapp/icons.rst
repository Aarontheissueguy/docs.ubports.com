.. _icons:

Suru Icons
==========

This is a list of all the Suru icons available for use with the `Icon <https://api-docs.ubports.com/sdk/apps/qml/Ubuntu.Components/Icon.html>`__ QML component.

Example Usage
-------------

.. code:: qml

        import Ubuntu.Components 1.3

        Icon {
            width: units.gu(5)
            height: units.gu(5)
            name: 'account'
        }
